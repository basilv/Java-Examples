Java Examples contains the Java code for the examples used in articles on the site Professional Software Development (basilv.com). The code for each article is placed into separate packages under com.basilv.examples.

Additional examples include:

  • The Ant build located in the root directory of the distribution includes an example of using Ivy for the article Using Ivy to Manage Build Dependencies.

Code is formatted using a non-standard coding convention to display better on the web site. Code is generally written with minimal comments or error-handling: it is intended for educational purposes and not necessarily for production use.
